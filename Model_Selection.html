So we get two different models. To compare the two regression models, we can use different criteria based on the trade-off between goodness-of-fit and model complexity. Some of the common criteria are:
  • R-squared: This is the proportion of variance in the response variable that is explained by the predictor variables. A higher R-squared indicates a better fit, but it can also increase with the number of predictors, even if they are not significant.
  • Adjusted R: This is a modified version of R-squared that penalizes the addition of predictors that do not improve the fit. A higher adjusted R-squared indicates a better fit, and it can be used to compare models with different numbers of predictors.
  • AIC: This is the Akaike information criterion, which measures the relative quality of a model based on the likelihood function. A lower AIC indicates a better fit, and it can be used to compare models with different numbers of predictors and different functional forms.
  • BIC: This is the Bayesian information criterion, which is similar to AIC but imposes a larger penalty for adding predictors. A lower BIC indicates a better fit, and it can be used to compare models with different numbers of predictors and different functional forms.
  • F-statistic: This is the test statistic for the overall significance of the regression model. A higher F-statistic indicates a more significant model, and it can be used to compare nested models, i.e., models that differ by one or more predictors.
  • p-value: This is the probability of obtaining a test statistic as extreme or more extreme than the observed one, assuming the null hypothesis is true. A lower p-value indicates a more significant model, and it can be used to compare nested models, i.e., models that differ by one or more predictors.
